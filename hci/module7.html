Module 7 — User Interface Design and Prototyping

Overview

This module explores UI design principles and prototyping approaches (Lo-Fi & Hi-Fi) and how to create interfaces that are both usable and visually clear.

7.1 What is UI Design?

UI design focuses on the look, feel, and interactivity of systems: layout, controls, navigation, and visual hierarchy to support UX.

7.2 Principles of Good UI

Consistency, feedback, simplicity, error prevention, flexibility, visibility, affordance, aesthetic minimalism — combine Shneiderman's rules and Nielsen's heuristics.

7.3 Components & 7.4 Prototyping

Components: input controls, navigation, information components, containers. Prototyping: Lo-Fi (wireframes, sketches) vs Hi-Fi (interactive mockups using Figma/Adobe XD).

7.5 UI Process & Tools

Process: research → sketch → prototype → evaluate → refine. Tools: Figma, Adobe XD, Sketch, Balsamiq, InVision.
