Module 5 — Storyboards and Design Evaluation

Overview

This module teaches how to create storyboards that visualize user interactions and introduces evaluation methods (expert-based and user-based) to assess design usability.

5.1 Storyboarding

Storyboard = sequence of panels showing a user's step-by-step interaction. Elements: title, persona, panels, narrative captions, context, outcome. Steps: define goal, identify persona, outline steps, sketch, add captions, revise.

5.2 Design Evaluation

Types: Expert-Based (heuristic evaluation, cognitive walkthrough) and User-Based (usability testing, think-aloud, field observation). Evaluation metrics: effectiveness, efficiency, satisfaction, learnability, error rate.

5.3 Iteration

Iterate: evaluate → refine → prototype → evaluate again until goals are met.
