Module 2 — The Human User: Cognition, Perception, and Ergonomics

Overview

This module explores human capabilities and limitations that underlie good interface design: perception, cognition (memory and mental models), and ergonomics.

2.1 Human Information Processing Model

Stages: Perception (senses), Cognition (processing, reasoning), Action (user response). Identify bottlenecks to design appropriately.

2.2 Human Perception & Gestalt Principles

Preattentive processing and Gestalt laws: Proximity, Similarity, Closure, Common Region. Design implication: group related items, use consistent styles.

2.3 Cognition and Memory

Mental models vs conceptual models; Sensory memory (<1s), Short-term/Working memory (~7±2 chunks, 10–15s), Long-term memory. Use recognition over recall.

2.4 Norman’s Principles (Gulfs)

Gulf of Execution (how to do it) and Gulf of Evaluation (what happened). Use affordances, signifiers, and clear feedback to bridge gulfs.

2.5 Ergonomics

Arrange controls for reachability, choose input devices to match tasks, prevent RSI, ensure proper font size & contrast, encourage breaks.
